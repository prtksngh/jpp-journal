Publish

When it is time to publish an issue, the Editor clicks "##editor.issues.publishIssue##" on the Live Issue page for the issue to be published, that is, the page with the issue's Table of Contents. Publishing an issue causes its Table of Contents to appear when Current is clicked by readers, and its cover, if there is one, to be appear on the journal's homepage. The issue that was previously Current goes to the Archives. If there is no cover, the Table of Contents will appear on the homepage, unless the homepage is occupied by an image or text that has been loaded in Setup by the Journal Manager to appear there. Publishing an issue also sends out an email notification to all registered Readers that includes the issue's Table of Contents and a link back to the journal's web site.
