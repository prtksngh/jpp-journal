Submissions

Once the Copyeditor logs in, clicking on the Copyeditor link in the User Home page leads to the Submissions queue. This queue lists those works which they have been asked to copyedit. Access is also provided to those submissions which they have completed in the past, via the Archive link.

To access the Copyediting page for a particular submission, the Copyeditor clicks on the work's title appearing in the Active Submission list.
